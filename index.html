TaskHive App Showcase

COP 4331 – Object Oriented Software Engineering · Florida Atlantic University · Fall 2025
Author: Adam Wallace · Project: TaskHive – Multi-List Task Manager

TaskHive is a Java desktop application that helps you manage multiple task lists with due dates, priorities, and categories.

Overview

TaskHive was created for students and busy users who need several separate lists instead of one giant to-do list. You can maintain different lists for:

  • Personal goals
  • Academic work
  • Professional tasks

Architecture & Design Patterns

TaskHive was designed using MVC plus Repository, Strategy, and Composite patterns.

MVC Layers

  • Model: Task, TaskListModel, TaskRepository
  • View: DashboardView, TaskListView
  • Controller: DashboardController, TaskListController

Main Patterns Used

Repository Pattern – Handles saving/loading lists from disk.

Composite Pattern – TaskListModel contains child Task objects.

Strategy Pattern – Filters and search behaviors.

How to Run TaskHive Locally

  1. Clone the GitHub repository.
  2. Open in IntelliJ or another IDE.
  3. Use Java 17+.
  4. Run the main class: TaskHiveApp.java.
