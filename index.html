TaskHive

Java desktop task manager for students and professionals.

Overview

TaskHive is a Java Swing desktop application that lets users create and manage multiple task lists. Lists can be grouped by category (personal, academic, professional), and each task can have due dates, priorities, and completion status to help users stay organized.

Key Features

  • Create, edit, and delete task lists and individual tasks.
  • Assign due dates and priorities, with visual cues for overdue tasks.
  • Filter and search lists by category, date, and urgency.
  • Local persistence so tasks are saved between sessions.
